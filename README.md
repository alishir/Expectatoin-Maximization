# Expectatoin-Maximization
Simple implementation of EM algorithm for one dimensional Gaussian mixtures using `breeze` and `scala`.

# Prerequisites 
- sbt 0.13+
- java

# How to run
- clone repo
- change directory to project root
- run `sbt` to get sbt console
- in sbt console `> run`
- see the results
